Decision Support System for Dengue Aid Eligibility - SAW Method

Data Science Project

Case Overview

Dengue fever (DBD) cases remain a serious public health issue in urban areas such as South Jakarta. The government often provides assistance to residents affected by dengue fever, but the selection process for recipients is often not based on objective data. Therefore, a web-based Decision Support System (DSS) is needed to help determine the eligibility of dengue assistance recipients in a fair and structured manner, using the Simple Additive Weighting (SAW) method based on several predefined criteria.

Scope & Goals

  • Collecting data of residents affected by dengue fever from sub-districts in South Jakarta, using forms provided by the relevant authorities.
  • Implementing the SAW method in a web-based system.
  • Evaluation criteria include severity level, income, number of dependents, and housing conditions.
  • The system output is a ranking list of eligible aid recipients.
  • Providing a system that facilitates officials in selecting aid recipients fairly and efficiently.
  • Delivering decision recommendations that are transparent and accountable.

Summary

This project resulted in a web-based decision support system application that implements the Simple Additive Weighting (SAW) method to determine the eligibility of dengue assistance recipients. The system is designed to input resident data, assign weights to each criterion, calculate the final score, and rank recipients based on eligibility. With this approach, the selection process becomes faster, more objective, and data-driven, while also facilitating decision-making for the relevant stakeholders.
