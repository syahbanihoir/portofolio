Market Basket Analysis - Apriori Algorithm with Excel

Data Analyst Project

Case Overview

Market Basket Analysis is a data analysis technique used to identify association relationships between items frequently purchased together by customers in a single transaction. This project aims to implement the Apriori algorithm on sales transaction data using Microsoft Excel to uncover purchasing patterns and generate insights that can support business decision-making, such as product bundling strategies, targeted promotions, and store layout optimization.

Scope & Goals

  • Utilizing sales transaction data in Excel format.
  • The dataset covers the period from February 1, 2024, to April 30, 2024, with a total of 1,024 records.
  • Applying the Apriori algorithm to discover association rules between products.
  • Presenting the analysis results on an interactive dashboard in tabular form.
  • Identifying product combinations that are frequently purchased together.
  • Generating association rules based on support, confidence, and lift values.

Summary

This project was carried out entirely using Microsoft Excel, starting from data cleaning, transaction matrix construction, to the calculation of support, confidence, and lift values. Without relying on programming tools, this project demonstrates that data analysis can be practically conducted using spreadsheet tools.

Methodology

2
