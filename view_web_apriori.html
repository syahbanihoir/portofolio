Market Basket Analysis with Machine Learning - Apriori Algorithm

Data Science Project

Case Overview

This system is designed to help business owners analyze consumer purchasing patterns based on sales transaction data. In the retail industry, understanding consumer buying patterns is crucial for marketing strategies and boosting sales. This project developed a web-based system using PHP that implements the Apriori algorithm to perform association analysis on sales transaction data. The primary goal of this system is to identify product combinations that are frequently purchased together (market basket analysis) to provide more targeted and effective business strategy recommendations.

Key features include:

  • Transaction Data Management: View transaction records and perform actions such as Edit, Delete, Input, and Upload transaction data in CSV or XLS formats.
  • Apriori Processing Module: Conduct Apriori Algorithm Analysis & Modeling by selecting a transaction date range and defining minimum support and minimum confidence values for the analysis.
  • Analysis Metrics & Dashboard: Display key Apriori analysis metrics along with a visualized dashboard of the results.
  • Apriori Process History: Maintain a history of past analyses with options to View, Print (PDF), or Delete the records.

Summary

This project is an implementation of the Apriori algorithm in the form of a web application. Users can upload sales transaction files and receive analysis results in the form of association rules along with evaluation metrics such as support, confidence, and lift, as well as an analysis dashboard presented in tables. This system is suitable for retail businesses that want to understand customer behavior and improve the efficiency of their sales strategies.
